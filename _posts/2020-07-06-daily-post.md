오늘은 삼성에서 일한 첫날

Oracle SQL Developer 에서 SQL sever로 connect 하기 위해서 여러가지 받은 정보로 로그인을 시도하였지만,
방화벽(?) 때문에 실패했다. 내일부터 제대로 된 근무가 시작될 것 같다.

1. CET(뭔지 모름) - main
2. ISAQ(뭔지 모름) - main
